Esse projeto foi desenvolvido com as seguintes tecnologias:

- Typescript
- React
- Styled Components

## 🛠️ Instalação do projeto

Instalação das dependências:
```
npm i
```

Comando para vizualizar a cobertura de testes da aplicação:
```
npm run test-coverage src
```
